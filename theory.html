1. Autoregressive Process: Definition and Model

1.1 The AR(p) Model

An Autoregressive process of order p, denoted AR(p), is a discrete-time stochastic process in which the current value is expressed as a linear combination of its p most recent past values plus a white noise excitation term. The governing difference equation is:

x(n) = -a1x(n-1) - a2x(n-2) - ... - apx(n-p) + v(n)
x(n) + a1x(n-1) + a2x(n-2) + ... + apx(n-p) = v(n)

where:

  • x(n): output of the AR process at time n
  • a₁, a₂, …, aₚ: autoregressive coefficients (real-valued)
  • p: model order — determines how many past samples influence the current output (memory depth)
  • v(n): driving white noise — zero mean, variance σ²v, independent of x(n−k) for k ≥ 1

Using the z-transform, with X(z) = Z{x(n)} and V(z) = Z{v(n)}, the system transfer function is:

H(z) = X(z) / V(z) = 1 / A(z)
A(z) = 1 + a1z-1 + a2z-2 + ... + apz-p

The AR model is an all-pole system — its transfer function has poles (determined by the AR coefficients) but no finite zeros other than at the origin.

1.2 Special Cases

Model Description and Behaviour
AR(0) x(n) = v(n). No memory — output equals the input directly. If v(n) is white noise, the output is white noise. AR(0) is a degenerate case, not inherently a white noise process.
AR(1) x(n) = −a₁x(n−1) + v(n). Single-pole system. If |a₁| < 1, produces an exponentially correlated process. Positive a₁ gives low-frequency dominant spectrum; negative a₁ gives high-frequency dominant spectrum.
AR(2) x(n) = −a₁x(n−1) − a₂x(n−2) + v(n). Can model oscillatory processes when poles are complex conjugate. Produces narrowband signals if poles are close to the unit circle.
AR(p), p ≥ 3 Higher-order models can produce multiple spectral peaks and represent processes with more complex temporal correlations. Used in EEG analysis to model multiple simultaneous brain rhythms.

1.3 Stationarity Condition

An AR(p) process is wide-sense stationary (WSS) if and only if all roots of the characteristic polynomial A(z) lie strictly inside the unit circle in the z-plane — i.e., |zᵢ| < 1 for all roots zᵢ of A(z) = 0. Equivalently, all poles of H(z) = 1/A(z) must be inside the unit circle. This is the only condition that needs to be checked before generating an AR process.

KEY RESULT
If any pole of H(z) lies ON or OUTSIDE the unit circle, the AR process is non-stationary (explosive or marginally stable). Signals will diverge or grow without bound. Always verify this condition before choosing AR coefficients for simulation.

For the important special case of AR(2), the stability conditions can be expressed directly in terms of the coefficients as the stability triangle:

(1) |a2| < 1
(2) a2 + a1 > -1
(3) a2 - a1 > -1

Example verification: a₁ = −1.5, a₂ = 0.7

(1) |0.7| = 0.7 < 1    CHECK
(2) 0.7 + (-1.5) = -0.8 > -1    CHECK
(3) 0.7 - (-1.5) = 2.2 > -1    CHECK → Stationary AR(2)

1.4 Power Spectral Density of an AR Process

The power spectral density (PSD) of a stationary AR(p) process is the Fourier transform of its autocorrelation function. For an AR(p) driven by white noise with variance σ²v, the PSD is:

Sxx(ω) = σ²v / |A(ejω)|²
A(ejω) = 1 + a1e-jω + a2e-2jω + ... + ape-jpω

This is a rational (all-pole) PSD. Its shape is entirely determined by the denominator polynomial A(ejω), which is the frequency response of the inverse filter. The key physical insight is:

  • Poles near the unit circle at angle θ₀ → sharp spectral peak at frequency f₀ = θ₀Fₛ / (2π)
  • Poles far from the unit circle → broad, flat spectrum (noise-like signal)
  • Complex-conjugate pole pairs → narrowband oscillatory signal in the time domain
  • Real poles → monotonically decaying or alternating autocorrelation (purely low-pass or high-pass)
KEY RESULT
The PSD formula Sxx(ω) = σ²v / |A(ejω)|² is the central result of AR spectral analysis. It explains why different AR coefficients produce different signal spectra, and is the theoretical basis for all parametric spectral estimation methods used in biomedical signal processing.

1.5 Autocorrelation Function and Yule-Walker Equations

The autocorrelation sequence rxx(k) = E[x(n)x(n+k)] of a stationary AR(p) process satisfies the Yule-Walker equations, which form the fundamental link between AR model parameters and the measurable autocorrelation structure of the signal:

rxx(k) + a1rxx(k-1) + a2rxx(k-2) + ... + aprxx(k-p) = 0,   k ≥ 1

In matrix form for k = 1, 2, …, p:

Rxxa = -r

where Rxx is the p×p Toeplitz autocorrelation matrix, a = [a₁, …, aₚ]T, and r = [rxx(1), …, rxx(p)]T.

The driving noise variance is recovered from:

σ²v = rxx(0) + a1rxx(1) + ... + aprxx(p)
σ²v = rxx(0) + aTr

These equations are solved efficiently using the Levinson-Durbin algorithm in O(p²) operations, compared to O(p³) for direct matrix inversion. In practice, when estimating AR parameters from data, the sample autocorrelation r̂xx(k) replaces the true autocorrelation, and the Yule-Walker equations become a least-squares estimation problem.

1.6 Effect of Model Parameters on Signal Behaviour

Parameter Effect on Time Series and Spectrum
Model order p (too low) Under-fitting: spectral peaks merged or smoothed, fine structure lost. Insufficient memory to model the true temporal correlations.
Model order p (too high) Over-fitting: spurious spectral peaks introduced, AR coefficients estimated with high variance. Effective order selection uses AIC or MDL criteria.
a₁ large positive (AR(1)) Single low-pass pole: slow, smooth, positively correlated time series. Autocorrelation decays slowly.
a₁ large negative (AR(1)) Single high-pass pole: rapidly alternating time series. Autocorrelation alternates in sign.
Complex-conjugate poles near unit circle Narrowband oscillatory signal — appears sinusoidal. Models EEG alpha waves, HRV spectral peaks.
Driving noise variance σ²v Scales the overall signal power uniformly. Does not affect spectral shape — only the height of the PSD.
Poles approaching unit circle Very long memory, large amplitude oscillations, approach to non-stationarity. Marginally stable.

1.7 Applications in Biomedical Signal Processing

  • EEG spectral analysis: AR models parameterise the power in delta (0.5–4 Hz), theta (4–8 Hz), alpha (8–13 Hz), beta (13–30 Hz), and gamma (>30 Hz) bands using low-order AR models (p = 8–20). AR-based PSD has better frequency resolution than the periodogram for short EEG epochs.
  • Heart rate variability (HRV): AR(16–20) models decompose HRV into low-frequency (LF: 0.04–0.15 Hz, sympathetic modulation) and high-frequency (HF: 0.15–0.4 Hz, parasympathetic/respiratory) components.
  • ECG P-wave and T-wave analysis: Low-order AR models fit to individual ECG segments capture morphological changes associated with ischaemia and arrhythmia.
  • EMG spectral fatigue analysis: The AR model tracks the progressive downward shift of EMG median frequency during sustained muscle contraction — a marker of muscle fatigue.
